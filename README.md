# ShopNest Backend

## Features

- JWT Authentication
- Categories CRUD
- Products CRUD
- Search
- Pagination
- Cart Management
- Order Management
- Reviews
- Admin Dashboard

## Tech Stack

- Node.js
- Express.js
- PostgreSQL (Supabase)
- Sequelize ORM
- JWT Authentication

## Installation

npm install

## Run

npm run dev

## API Docs

http://localhost:5000/api-docs
